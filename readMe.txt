Pour lancer le site, il faut importer la base de données (le nom d'utilisateur est à modifier dans la fonction connectToBDD de "Bibli.php" & "Bibli_admin.php"
Ensuite il faut juste lancer /php/index.php pour le mode utilisateur, et /php/index_admin.php pour le mode administrateur 
Pour changer le PREFIXE de la base, il faut le changer dans Bibli.php, Bibli_admin.php et upload.php
